# BRUTE PRO – Hydra Automation Tool

BRUTE PRO is a Python-based interface that automates common Hydra brute-force 
commands (SSH, FTP, RDP, SMB, HTTP, MySQL, ZIP cracking, and more).

## Requirements

### Linux
sudo apt install hydra python3 python3-pip -y 

or

sudo dnf install hydra python3 python3-pip -y 

### Windows (via WSL)
wsl --install -d kali-linux
sudo apt install hydra python3 python3-pip -y

### Python Dependencies
pip install -r requirements.txt

## Run
python3 brutepro.py
